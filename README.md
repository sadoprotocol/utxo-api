# UTXO API

## Requirement

1. NodeJS
2. MongoDB


## Setup

Set the following

```sh
# Set the environment to proper values
$ cp dotenv .env

# Install dependencies
$ npm install
```

## Usage

```sh
# Start
$ npm start
```

## Available API endpoints

Find it in our [docs](https://github.com/sadoprotocol/utxo-api/tree/2a06b053759d9666c471d92708c35ab152180cf0/docs).
