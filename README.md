# UTXO API

## Requirement

1. NodeJS
2. MongoDB


## Setup

Set the following

```sh
# Set the environment to proper values
$ cp dotenv .env

# Install dependencies
$ npm install
```

## Usage

```sh
# Start
$ npm start
```

## Supported Explorer

- [UTXO Parser](https://github.com/sadoprotocol/utxo-parser)
- [BlockCypher](https://www.blockcypher.com)
